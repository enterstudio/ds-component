Const parseImage

  • parseImage(value: string): ParsedImage
    • Defined in index.ts:102

    Parses a '\u00a0\u00a0' delimited string into component parts. If any parts are missing, they will be returned as undefined.

    Usage:

    const myImage = parseImage('originalurl.com\u00a0\u00a0proxiedurl.com\u00a0\u00a0alt text');

expect(myImage).toEqual({
  originalUrl: 'originalurl.com',
  proxiedUrl: 'proxiedurl.com',
  altText: 'alt text',
});
